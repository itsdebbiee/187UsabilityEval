No Breadcrumbs

This site doesn’t have breadcrumbs. This gives the user more responsibility to remember his/her actions and the information on previous pages. There are no indicators of where the user is besides the browser address or perhaps the homepage of the respective site. Without breadcrumbs, this site is harder to navigate and things are not easily retrievable.

Logo Change

The logo changes when the user navigates through the site. The user cannot return back to the homepage from most of the subsites. The university’s brand gets lost and mixed up. Given the amount of information presented on the site, the user is still not presented with a visible and recognizable logo or idea of what the school is like. The logo should be some sort of statement, but since there is no consistency, it is hard for a user to recognize which page they are on, especially if the logos are different from page to page.
