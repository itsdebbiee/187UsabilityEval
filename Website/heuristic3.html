No "Emergency Exit"

There is not a lot of user control regarding navigation within the website. When we’re on individual pages of the site, it is difficult to return to the original home page. We expect to click on the logo, but that only takes us back to the home page for that individual academic site, not the original GSU home site. After browsing the site, users can typically figure out that returning to Georgia State Home page would require you to click on the top navigation bar, not the logo anymore.

Can’t Go Back

After clicking into separate sites, such as the PAWS login page, it is difficult to return back to the Home Page or even the previous page. There are no breadcrumbs, which restricts the user from clicking back. Even using the back browser won’t take the user back to the original home page. To go back, users would have to manually click out of the PAWS login page or double click the back browser button. Another example, is the campus maps page. Again, GSU’s website doesn’t give the user much freedom in terms of navigating from page to page.
