Required Field

GSU’s site does a good job providing error messages when filling out the account registration. This allows the user to see which field is required to create an account and continue with the application process.

Panther Predictor Tool

The “Panther Predictor” tool calculates the SAT/GPA/ACT scores for the user but it does not inform usersr if they are missing any required fields. Instead, the user will be directly informed that their admissions is denied, because they do not meet the minimum requirements for admission.
